# Seteuk Workflow

학생 및 수업 관련 자료가 담긴 폴더 하나로 세특(교과세부능력 및 특기사항) 작성의 전 과정을 자동으로 처리하는 플러그인입니다. 모든 교과, 모든 학년에 적용 가능한 범용 워크플로우입니다.

## 개요

폴더 안에 형식이 제각각인 학생 자료(PDF, hwpx, docx, pptx, xlsx/csv, 이미지, txt 등)와 교사의 평가/관찰 기록을 모아두고 이 플러그인을 실행하면, 필요한 정보를 질문으로 확인한 뒤 아래 세 단계를 이어서 자동으로 진행합니다.

1. **자료 정리** — 원자료를 표준 엑셀로 정리
2. **초안 작성** — 정리된 자료를 근거로 학생별 세특 문단 작성 (학생 수가 많으면 서브에이전트 병렬 처리)
3. **최종 점검** — 절대 기재금지, 관찰자 시점, 서식 규정, 분량, 개별화, 자연스러움까지 검수하고 마무리 (학생 수가 많으면 서브에이전트 병렬 처리)

## 병렬 처리

초안 작성과 최종 점검 단계는 대상 학생 수에 따라 자동으로 서브에이전트 개수를 정해 병렬로 처리합니다.

| 학생 수 | 서브에이전트 수 |
|---|---|
| 1~3명 | 사용 안 함(직접 처리) |
| 4~6명 | 2개 |
| 7~10명 | 3개 |
| 11~15명 | 4개 |
| 16~20명 | 5개 |
| 21명 이상 | 6개(상한) |

각 서브에이전트는 담당 학생의 자료만 받아 독립적으로 작성/점검하며, 결과는 오케스트레이터가 취합해 학생 순서대로 정리합니다. 자세한 기준은 `skills/seteuk-workflow/references/05-parallel-dispatch.md`를 참고하세요.

## 사용법

Cowork(또는 Claude)에 이 플러그인을 설치한 뒤, 학생/수업 자료가 담긴 폴더를 연결하고 아래와 같이 요청하면 스킬이 트리거됩니다.

- "세특 워크플로우 시작해줘"
- "이 폴더로 세특 전체 작업 해줘"
- "자료 정리부터 세특 작성, 점검까지 한번에 해줘"

실행되면 먼저 교과목명, 학년, 학기, 나이스 바이트 제한, 대상 학생 범위, 세특 강조점(선택)을 질문으로 확인한 뒤 3단계를 자동으로 진행하고, 최종적으로 `세특_최종.xlsx` 파일을 전달합니다.

## 주의 사항

- 나이스 바이트 제한과 기재금지 세부 기준은 학년도마다 소폭 개정됩니다. 최종 확정은 반드시 해당 학년도 나이스 공지나 학교생활기록부 기재요령으로 재확인해야 합니다.
- 근거가 부족하거나 판단이 애매한 부분은 임의로 채우지 않고 사용자에게 되묻도록 설계되어 있습니다.
- pdf, hwpx, docx, pptx, xlsx 형식의 자료 추출에는 해당 형식을 다루는 별도 스킬(pdf, hwpx, docx, pptx, xlsx)이 함께 설치되어 있으면 더 안정적으로 작동합니다.
