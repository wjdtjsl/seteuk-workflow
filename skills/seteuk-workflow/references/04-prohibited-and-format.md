# 세특 검수 상세 기준 (기재금지, 서식, 분량, 개별화)

학교생활기록부 기재요령(교육부, 시도교육청 공통 기준)에 근거한 세특(교과학습발달상황 세부능력 및 특기사항, 개인별 세특 포함) 검수 기준이다. 3단계(점검)가 참조 문서로 로드한다. 교과, 학년과 무관하게 동일하게 적용한다.

주의: 학교생활기록부 기재요령은 학년도마다 세부 조항이 소폭 개정된다. 아래 항목은 통상적으로 여러 학년도에 걸쳐 공통으로 유지되어 온 핵심 기준이다. 최종 확정은 반드시 해당 학년도 나이스 공지로 재확인해야 한다.

## 1. 절대 기재 금지 항목

문장에 아래 내용이 조금이라도 포함되어 있으면 삭제하거나 관찰 가능한 교내 활동 서술로 대체한다. 삭제만으로 문장이 어색해지면 사용자에게 대체할 교내 활동 근거를 되묻고, 임의로 지어내지 않는다.

| 구분 | 금지 내용 | 감지 키워드 예시 |
|---|---|---|
| 논문 | 논문 투고, 등재, 발표 사실 (학회지, 소논문, 저널 게재 등) | "논문", "소논문", "학회지 게재", "저널에 발표" |
| 도서출간 | 학생의 저서 출간 사실 | "출간함", "저서를 발간", "책을 출판" |
| 지식재산권 | 특허, 실용신안, 상표, 디자인 출원 또는 등록 사실 | "특허 출원", "특허 등록", "실용신안" |
| 인증시험 | 교내외 어학, 자격 인증시험 참여 사실이나 성적, 응시 사실 (국가기술자격은 제외) | "토익", "토플", "오픽", "한자능력검정", "컴활 몇급", "인증시험 응시" |
| 모의고사 성적 | 모의고사, 전국연합학력평가 관련 성적, 석차 | "모의고사 등급", "전국연합학력평가 점수" |
| 교외 대회 실적 | 교외 대회 참가 사실, 수상 실적(대회명, 순위, 상장명) | "대회 참가", "대회 수상", "상 수상", "전국대회", "교육청 주최 대회" |
| 해외활동 | 해외 어학연수, 해외 봉사활동, 교환학생 등 | "해외연수", "해외봉사", "교환학생" |
| 부모, 가족 배경 | 부모(친인척)의 사회경제적 지위를 암시하는 내용 | "아버지가", "부모님 직업", "집안 형편" |
| 장학 관련 | 장학생 선발, 장학금 수혜 사실 | "장학생", "장학금" |
| 특정 기관, 상호명 | 특정 대학명, 국제기구명(유네스코, WTO, OECD, UN, 통계청 등), 상호명(구글, 네이버, 유튜브, 삼성, 카카오톡 등), 학원명, 강사 실명 | "○○대학교 진학을 목표로", "유네스코", "구글", "○○학원", "메가스터디" |
| 자격증 | 국가기술자격을 제외한 자격증, 인증 취득 사실 | "자격증 취득", "인증 획득" |

**교정 전략**: (1) 해당 문구를 제거한다. (2) 그 문구가 뒷받침하려던 역량이나 특성을 교내 수업 활동(발표, 실습, 프로젝트, 산출물)으로 재서술할 수 있는지 문맥에서 확인한다. (3) 대체할 교내 활동 근거가 문장에 없다면 사용자에게 "이 부분을 교내 활동 중 어떤 근거로 대체할까요?"라고 되묻는다.

## 2. 서식 규정 (나이스 입력 요건)

1. **명사형 종결 강제**: 모든 문장은 "~함", "~보임", "~수행함", "~조사함" 등 명사형으로 종결한다. "~했다", "~한다", "~합니다" 등으로 끝나면 명사형으로 변환한다.
   - 예: "발표를 진행했다" → "발표를 진행함"
2. **주어 생략**: 문장마다 "학생은", "OOO는", "본인은" 같은 주어를 반복하지 않는다. 꼭 필요한 경우가 아니면 생략한다.
   - 예: "학생은 실험을 설계함" → "실험을 설계함"
3. **특수기호 사용 금지**: 가운데점(·), 물결표(~), 괄호를 이용한 병기, 한글 문자표의 특수기호를 모두 금지한다. 쉼표나 "와/과"로 대체한다.
   - 예: "물리·화학 개념을" → "물리, 화학 개념을" 또는 "물리와 화학 개념을"
4. **날짜 형식**: 0000.00.00.-0000.00.00. 형식을 지킨다. 물결(~)이 아닌 하이픈(-)으로 연결하고, 8자리 숫자와 마침표를 반드시 포함한다.
   - 예: "2026.03.02.~2026.07.18." (틀림) → "2026.03.02.-2026.07.18." (맞음)
   - 1, 2, 3학년 1학기 재임기간은 실제 개학일과 무관하게 매년 3월 1일부터로 통일한다.
5. **도서명 표기**: 단순 독후감 활동이면 도서명을 입력할 수 없다. 발표, 토론, 보고서, 콘텐츠 제작 등에 실제로 활용된 경우에만 (도서명, 저자명) 또는 도서명 단독으로 입력할 수 있다.
   - 예: "『이기적 유전자』를 읽고 감상문을 작성함" (도서명 입력 불가, 단순 독후감) → "생명과학 개념을 다룬 도서를 읽고 감상문을 작성함"
   - 예: "『이기적 유전자』(리처드 도킨스)의 이기적 유전자 개념을 활용해 토론 자료를 제작하고 발표함" (도서명 입력 가능, 토론과 발표에 활용됨)

## 3. 분량(바이트) 제한

나이스는 한글 1자를 3바이트, 숫자와 영어와 띄어쓰기와 엔터는 1바이트로 계산한다. 과목별 세특은 통상 500자(약 1500바이트)가 여러 학년도에 걸쳐 유지되어 왔지만, 영역(개인별 세특, 진로활동, 자율/동아리)이나 학년도마다 조금씩 바뀔 수 있다.

- 과목별 세부능력 및 특기사항: 통상 500자 (한글 기준 약 1500Byte)
- 개인별 세부능력 및 특기사항: 학년도 개정에 따라 변동
- 진로활동 특기사항: 학년도 개정에 따라 변동
- 자율활동, 동아리활동 특기사항: 통상 500자 내외 (교육청, 학년도별 상이)

특수문자와 개행도 바이트에 포함되므로 눈대중보다 `scripts/byte_counter.py` 같은 계산 스크립트로 확인하는 것이 정확하다. 결과에는 항상 "정확한 최신 기준은 나이스 공지로 재확인 권장"을 덧붙인다.

## 4. 개별화와 구체성 (규정 통과 이후 매력도)

규정 준수는 최소 조건일 뿐이다. 아래 기준으로 설득력을 점검하되, 모두 원문에 있는 사실 범위 안에서 표현을 개선하는 것이며 없는 활동이나 성과를 창작하지 않는다.

1. **공통문구 지양**: 다른 학생에게 그대로 붙여도 어색하지 않은 뻔한 문장인지 확인한다.
2. **구체성**: "열심히 함", "성실히 참여함" 같은 상투적 표현을 활동명, 산출물, 수치(실험 횟수, 발표 주제, 제작물 이름 등)로 대체할 수 있는지 확인한다.
3. **서사 구조**: 계기(문제의식) → 활동(구체적 행동) → 결과(산출물, 발견) → 확장(후속 탐구, 연계 활동)의 흐름이 드러나는지 확인한다. 나열식 문장만 반복되면 순서 재배치를 제안한다.

## 5. 우선순위

세특 검수는 아래 순서로 처리한다.

1. 절대 기재 금지 항목 (1항) — 규정 위반, 최우선 수정
2. 관찰자 시점 — 신뢰도 문제
3. 서식 규정 (2항) — 나이스 입력 요건, 필수 수정
4. 분량 제한 (3항) — 초과 시 필수 수정
5. 개별화와 구체성 (4항) — 위 1~4를 만족한 뒤 제안
6. 자연스러움 마무리 — 모든 규정 수정이 끝난 뒤 마지막 단계
